# C Conceptual Programs

This repository contains basic **C language conceptual programs** designed to understand core programming fundamentals.

## 📌 About
The programs in this repository focus on:
- Strengthening C programming concepts
- Practicing beginner to intermediate level problems

## 📂 Contents
This repository includes programs related to:
- Basic input/output
- Conditional statements
- Loops
- Functions
- Arrays
- Pointers
- Structures
- Conceptual logic programs

##  How to Compile and Run
Use the following commands to compile and execute a program:

gcc filename.c -o Myexe

To Run

./Myexe
